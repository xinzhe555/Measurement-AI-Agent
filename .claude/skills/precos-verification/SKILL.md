## ZOA 解析公式（已驗證，與 C 軸完全獨立）

```
dX = 0
dY = −ZOA × sin(θa)
dZ =  ZOA × (1 − cos(θa))
```

## 學長公式（對照用）

學長公式（`公式計算詳述.docx`）：
```
P₂ = A × ((C × (P₁ − (Q+S)) + (Q+S)) − M) + M

Q = [XOC, YOC, 0]
S = [0, YOA, 0]
M = [0, YOA, ZOA+L]
```

**重要差異**：學長公式使用平移向量（無角度誤差項），本研究使用 ISO HTM（含角度誤差矩陣），兩者在僅有位移誤差且 tool_length=0 時等效。
